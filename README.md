# Secure-E-Commerce-Web-Application
# 📌 Project Overview
Developed a full-stack e-commerce web application enabling secure online transactions with OTP-based verification. Implemented an admin dashboard to monitor user activity, analyze transactions, and detect potentially fraudulent behavior. Focused on integrating security practices with real-world web application workflows.

# 🚀 Key Features
* **🔐 Secure web-based system using Flask**
* **💳 Transaction processing and monitoring**
* **🚨 Detection of suspicious or abnormal transactions**
* **📊 Structured data handling with SQLite**
* **🖥️ Clean and simple user interface**
* **⚡ Lightweight and fast execution**
* **🛠️ Tech Stack**
# Category	Technology Used
Backend	Python (Flask)
Frontend	HTML, CSS, JavaScript
Database	SQLite
Architecture	MVC (Flask-based)
